About This Project

An end-to-end AI pipeline for detecting Fake & Spam News using NLP and Machine Learning.

🎯 Project Overview

This project builds a complete AI-powered Spam News Detector. It uses Natural Language Processing (NLP) to preprocess raw news text, TF-IDF vectorization to extract features, and a trained Machine Learning classifier to predict whether a given news article is Real or Fake/Spam.

🔬 Tech Stack

  • Python 3.x – Core language
  • Flask – Web framework & API server
  • scikit-learn – ML models (PAC, LR, NB)
  • NLTK – Stopwords, stemming
  • TF-IDF Vectorizer – Feature extraction
  • Pandas / NumPy – Data handling
  • Matplotlib / Seaborn – Visualization
  • HTML5 / CSS3 / JS – Frontend

⚙️ Pipeline Steps

  1. Load labeled dataset (fake.csv + true.csv)
  2. EDA — shape, nulls, class distribution
  3. Text cleaning — lowercase, remove URLs/HTML/special chars
  4. Stopword removal + Porter Stemming
  5. TF-IDF Vectorization (max 5000 features, bigrams)
  6. Train Passive Aggressive Classifier (primary)
  7. Evaluate — accuracy, confusion matrix, F1-score
  8. Save model via joblib
  9. Serve via Flask REST API
  10. Display result in interactive web UI

📂 Dataset

This project is designed for the Kaggle Fake News Dataset which contains two CSV files:

  • data/fake.csv – Fake/spam news articles (label = 0)
  • data/true.csv – Real news articles (label = 1)

If the dataset is not present, the trainer auto-generates synthetic demo data so you can still test the full pipeline.
