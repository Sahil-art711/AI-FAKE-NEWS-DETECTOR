# 157-AI-FAKE-NEWS-DETECTOR
THIS APP IS USED FOR DETECTING THE FAKE NEWS AND GIVE THE 100 PERCENT GENUINE MATERIAL ABOUT THAT PARTICULAR NEWS

📌 Overview
The Fake News Detection App is an AI-powered web application designed to analyze news content and determine its authenticity. The app evaluates a given piece of news and provides a clear verdict:
✅ Real
❌ Fake
⚠️ Misleading
This project helps users quickly identify unreliable or deceptive information, promoting awareness and responsible content consumption.

🚀 Features
🔍 Analyze news articles or headlines
🤖 AI/ML-based prediction model
📊 Instant verdict: Real, Fake, or Misleading
🌐 Simple and user-friendly web interface
⚡ Fast and efficient processing

🛠️ Tech Stack
Frontend: HTML, CSS
Backend: Python (Flask)
Machine Learning: Scikit-learn / NLP
Dataset: Fake & Real News Dataset
